AngularJS XLTS Information

Prior to January 1st 2021 AngularJS XLTS will be forked from the latest AngularJS 1.7.x. On January 1st 2021, AngularJS XLTS will become available to our customers.

We will continue the exclusive focus on providing fixes to bugs that satisfy at least one of the following criteria:

  • A security flaw is detected in AngularJS XLTS
  • One of the major browsers releases a version that will cause current production applications using the AngularJS XLTS to stop working
  • The jQuery library releases a version that will cause current production applications using AngularJS XLTS to stop working

Applications that use NPM and are based on the latest AngularJS 1.7.x will be able to seamlessly move over to AngularJS XLTS by updating their package.json dependencies.
