AngularJS Long Term Support

On July 1st 2018, AngularJS entered a limited, 3 year Long Term Support1 period.

The Angular team now focuses, exclusively, on providing fixes to bugs that satisfy at least one of the following criteria:

  • A security flaw is detected in the 1.7.x branch of AngularJS
  • One of the major browsers releases a version that will cause current production applications using AngularJS 1.7.x to stop working
  • The jQuery library releases a version that will cause current production applications using AngularJS 1.7.x to stop working

The Angular team will update the AngularJS 1.2.x branch if and only if a new, severe security issue is discovered.

1AngularJS LTS is provided by the Angular team and not the XLTS.dev team.
